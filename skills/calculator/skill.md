# Calculator Skill

**ID:** calculator  
**Version:** 1.0.0  
**Author:** Akasha  
**Category:** utility  

## Description

Math and unit conversions. The agent can perform calculations and look up conversions via tools.

## Capabilities

- Perform arithmetic operations (addition, subtraction, multiplication, division, powers, roots)
- Evaluate complex mathematical expressions
- Unit conversions (length, weight, temperature, area, volume, speed, etc.)
- Currency conversion (via web search for live rates)
- Scientific calculations (trigonometry, logarithms, etc.)

## Installation

To install this skill, send the following message to your Akasha agent:

```
Install the calculator skill from https://github.com/azerothl/Akasha_skills/tree/main/skills/calculator
```

## Usage Examples

- "Calculate 15% tip on a $47 bill"
- "Convert 100 miles to kilometers"
- "What is the square root of 144?"
- "How many ounces are in 2.5 pounds?"
- "Convert 37°C to Fahrenheit"
